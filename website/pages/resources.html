The Public Resources

Uniworld provides some resources for developers and applications to integrate with the Unichain network. As the public resources, all the nodes are accessed by publicity. We have the rate limit and cannot guarantee the availability if the nodes are over accessed. For stability and production use, please run your own nodes. The guides to run the blockchain nodes can be found here

How to use the public resources

The public resources can be used for

  • Construct the UniChainJs library, check out this link
  • Call API from nodes, check out this API documents
  • Config for UniWallet extension (network), CLI Wallet and other purposes

Developers may need some UNW for testing purposes. They can claim the faucet from testnet by this link. Please note that UNW in testnet has no value and can be reseted at any time.
